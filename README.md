# SERVANT Manager
SERVANT Manager is the main GUI of SERVANT.  

## Build Dependencies
* Qt 5.8.0 or later  
* servant-base-lib  
* dependencies of servant-base-lib except boost

## How to build
* put libSERVANTBase.a in ./../../servant-base-lib/cmake-build-debug/  
* use Qt with MinGW-w64(require filesystem support)
